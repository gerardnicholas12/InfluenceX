## 📖 Overview

**InfluenceX** is a responsive **Single Page Application (SPA)** developed using **HTML, CSS, and JavaScript**. The platform is designed to bridge the gap between **brands** and **content creators** by providing a modern interface where brands can discover talented creators and creators can showcase their profiles and collaboration opportunities.

The application offers smooth navigation between sections without loading multiple pages, creating a seamless user experience.

---

## 🎯 Project Objectives

- Connect brands with suitable content creators.
- Provide an engaging and responsive user interface.
- Demonstrate modern front-end web development concepts.
- Showcase a clean and professional landing page for influencer marketing.

## 🖥 Application Sections

Introduces the platform with a visually appealing hero section and call-to-action.
